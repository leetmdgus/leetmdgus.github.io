## 코레일톡

+ 기능:
  + 역명검색기능(출발 / 도착 바꿔서 바로 선택가능)
  + 날짜 시간 입력
    + 오늘을 기준으로 하는 일주일 달력 제공
  + 열차조회결과
    + 일반식 특실 좌석 현황 조회 및 가격 확인
  + 열차표확인
    + 출발 도착점
    + 출발 도착시간
    + 열차이름
    + 객실위치
    + 발권일시
  + 실시간 운행 정보
